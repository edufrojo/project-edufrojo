## Despliegue

Descargar repositorio:

```sh
git clone https://github.com/edufrojo/project-edufrojo.git
```

Instalar dependencias:

```sh
npm install
```

Arrancar servidor:

```sh
npm run start-prod
```

Acceder a través del navegador a `localhost:3000`
